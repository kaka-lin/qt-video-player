# OpenCV and QML's VideoOutput

Example code of show video on QML ([VideoOutput ](https://doc.qt.io/qt-5/qml-qtmultimedia-videooutput.html)) use OpenCV

## Require

- Qt5.10+
- CMake & Ninja

## Build and Run

### 1. qmake

```bash
$ ./build.sh
```

### 2. CMake

```bash
$ ./cmake-build.sh
```

### 3. Run with Docker

Please see this [doc](../doc/run_with_docker.md)
